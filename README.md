This responsive Website is built with React, HTML and CSS. This website allows users to buy plants and flowers. 

## Demo Website
Demo website : https://envirogreen.herokuapp.com/

## To run this project:
1. Clone this repo
2. Run `npm install`
3. Run `npm start`
